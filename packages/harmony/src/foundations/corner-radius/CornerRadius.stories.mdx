## Overview

Corner radius variables are used to give sharp edges a more subtle, rounded effect and impact the overall look and feel of Harmony components.

</Heading>

export function CornerRadiusExample(props) {
  const { cornerRadius } = props
  const theme = useTheme()
  const cornerRadiusValue = theme.cornerRadius[cornerRadius]
  return (
    <Flex
      alignItems='center'
      justifyContent='center'
      direction='column'
      gap='s'
      h={108}
      w={108}
      css={{
        borderRadius: cornerRadiusValue,
        border: '1px solid #000'
      }}
    >
      <Text>{cornerRadius}</Text>
      <Text>{cornerRadiusValue}px</Text>
    </Flex>
  )
}

    ))}

</Unstyled>

## Usages and examples

| Name       | Description                                                                                                                                              |
| ---------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| xs - 2px   | Used for tags and extra small elements.                                                                                                                  |
| s - 4px    | Used for buttons and artwork on track tiles.                                                                                                             |
| m - 8px    | Used as the default for cards, containers and large artwork.                                                                                             |
| l - 12px   | Used for containers containing cards such as tables and track pages.                                                                                     |
| xl - 16px  | Accent to be used sparingly                                                                                                                              |
| 2xl - 32px | Used to round off the smaller edge to a half circle, creating a pill button, or to create a circle (full) as the background to an icon or other graphic. |
