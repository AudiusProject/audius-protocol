## Register a Node

1. Navigate to the [Audius Protocol Dashboard Nodes Tab][nodes-tab]
2. Click the blue "Connect Wallet" button on the upper right

<figure className="screensnippet-wrapper">
  <img src={useBaseUrl('/img/register-node/wallet-connect-button.png')} />
  <figcaption>Wallet Connect Button</figcaption>
</figure>

3. Select your web3 wallet in the wallet selection modal and sign in

<!-- prettier-ignore -->
<figure className="screensnippet-wrapper">
  <img src={useBaseUrl('/img/register-node/wallet-selector.png')} />
  <figcaption>Be sure to connect with the `Service Provider Owner Wallet Address` wallet, not a delegate address wallet</figcaption>
</figure>

4. Click the "Register New Service" button and complete the form.

<!-- prettier-ignore -->
<figure className="screensnippet-wrapper">
  <img src={useBaseUrl('/img/register-node/register-node-modal.png')} />
  <figcaption>Select the Node Type to Register</figcaption>
</figure>

5. Select the Node Type, `Discovery Node` or `Content Node`
6. Enter a fully qualified domain name in the url bar `Node Endpoint` field
7. Enter a token amount in the `Stake` field
8. Click the `Register <Service type> Node` button
