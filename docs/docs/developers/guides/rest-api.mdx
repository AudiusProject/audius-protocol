The Audius API is entirely free to use. We ask that you adhere to the guidelines in this doc and
always credit artists.

The Audius REST API lets you query, stream, and search for tracks, users & playlists across the
network.

Try it out!

```bash
curl https://discoveryprovider.audius.co/v1/tracks/trending?app_name=ExampleApp
```
