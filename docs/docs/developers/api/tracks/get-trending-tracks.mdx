## Get Trending Tracks

Gets the top 100 trending (most popular) tracks on Audius

<Tabs
  block={true}
  defaultValue="request"
  values={[{"label":"Request","value":"request"},{"label":"Response","value":"response"}]}
>
<TabItem value="request">

```bash
GET https://audius-discovery-3.theblueprint.xyz/v1/tracks/trending?app_name=EXAMPLEAPP HTTP/1.1
```

> Path Parameters

| Param    | Type     | Required | Note                                           | Available values                |
| -------- | -------- | -------- | ---------------------------------------------- | ------------------------------- |
| genre    | string   | false    | Filter trending to a specified genre           |
| time     | string   | false    | Calculate trending over a specified time range | `week` `month` `year` `allTime` |
| app_name | _string_ | false    | Your App Name                                  |

</TabItem>
