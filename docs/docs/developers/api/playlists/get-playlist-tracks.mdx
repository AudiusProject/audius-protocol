## Get Playlist Tracks

Gets tracks within a playlist.

<Tabs
  block={true}
  defaultValue="request"
  values={[{"label":"Request","value":"request"},{"label":"Response","value":"response"}]}
>
<TabItem value="request">

```bash
GET https://audius-discovery-3.theblueprint.xyz/v1/playlists/DOPRl/tracks?app_name=EXAMPLEAPP HTTP/1.1
```

> Path Parameters

| Param       | Type     | Required | Note          |
| ----------- | -------- | -------- | ------------- |
| playlist_id | _string_ | true     | A playlist ID |
| app_name    | _string_ | false    | Your App Name |

</TabItem>
